# AI Middle School Maze
By: Haneen Odeh

## Maze Theme
This maze explores the topic of **AI and middle school education**. 
The directories represent different stages of thinking about how artificial intelligence can be used in classrooms.

## Goal
Navigate through the directories to reach the final solution.

## Helpful Commands

You may find the following command line tools helpful:

- `ls` or `dir` — list files and directories
- `cd` — move between directories
- `cat` — read file contents
- `ls -la` — reveal hidden files or directories

## Hint
Some directories may contain clues that guide you to the next location.

Hidden files or directories may exist.
